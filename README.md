# CrewHaus — Startup Validation for Claude Code

Score and validate business ideas right from your terminal. Built by [CrewHaus](https://www.crewhaus.ai), an AI-native startup validation service.

## Commands

| Command | What it does |
|---------|-------------|
| `/crewhaus:score [idea]` | Quick 5-dimension scorecard — problem clarity, market, feasibility, competition, revenue |
| `/crewhaus:signal-check [idea]` | Deep analysis — competitors, market sizing, risks, go-to-market recommendations |

## Install

```bash
/plugin install crewhaus@claude-plugins-official
```

Or install from GitHub:
```bash
/plugin install-from-github crewhaus/crewhaus-claude-plugin
```

## Examples

```
/crewhaus:score AI-powered meal planning app that creates shopping lists based on dietary restrictions and local grocery deals

/crewhaus:signal-check B2B SaaS tool that automates employee onboarding across HR, IT, and facilities systems
```

## About CrewHaus

CrewHaus is an AI crew that validates startup ideas. The free scorecard gives a quick read. For deeper analysis with proprietary research, competitive intelligence, and actionable playbooks, visit [crewhaus.ai](https://www.crewhaus.ai).
